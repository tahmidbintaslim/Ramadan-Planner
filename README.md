# Ramadan Planner (BD-first)

A Bengali-first Ramadan planning and tracking website built with Next.js 16, Supabase, and shadcn/ui. Secondary English support for expats.

## Quickstart

- Install: `npm install`
- Dev: `npm run dev`
- Lint: `npm run lint`
- Build: `npm run build`
- Test: `npm run test`

## Core Modules

- Auth (Supabase): email + OAuth
- Daily Planner: ayah/hadith/dua + salah/quran/checklist + journal
- Dynamic prayer/sehri/iftar timing + Hijri date per timezone
- Reminders + announcements
- Calendar sync (Google + ICS)
