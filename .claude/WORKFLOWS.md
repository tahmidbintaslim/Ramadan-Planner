# Workflows

## Standard Feature Flow

1. Update spec/docs if needed
2. Implement minimal code changes
3. Add tests (unit and/or E2E)
4. Run `npm run lint`
5. Run `npm run test`
6. Ensure i18n keys exist BN/EN
7. Verify perf impact (bundle/caching)

## Output Format for Agent

- Summary
- Files changed
- Risks/edge cases
- Test checklist
