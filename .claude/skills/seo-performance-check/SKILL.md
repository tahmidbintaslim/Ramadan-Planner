# Skill Instructions

1. Validate localized metadata for public pages (BN/EN)
2. Add/verify robots + sitemap
3. Ensure private routes are noindex
4. Check bundle size and client components creep
5. Recommend caching points (daily content, timings)
6. Output: P0/P1/P2 fix list + expected impact
7. Run `npm run lint` and `npm run build`
