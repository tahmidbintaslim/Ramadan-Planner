# Skill Instructions

1. Create page route under App Router
2. Use Server Components by default; Client only for interactivity
3. Use shadcn/ui primitives; Tailwind utilities only
4. Add BN/EN translation keys for all user strings
5. Add loading/empty/error states
6. Update Playwright E2E if the flow is user-visible
7. Run `npm run lint`
